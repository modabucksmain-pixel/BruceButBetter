Prefer esptool?

esptool.py --chip esp32s3 --port COMx write_flash 0x0 Bruce-esp32-s3-devkitc-1-psram.bin

// 05 — help

FAQ

Before you flash.

Which browser is required?

Desktop Chrome or Edge — they support Web Serial. Firefox and Safari do not. Mobile browsers will not work.

Flashing won't start / no port appears

Hold BOOT while plugging in (or BOOT+RESET) to force download mode. Install the CH34x or native-USB serial driver, and use the left USB-C port.

Do I need every module populated?

No. Modules are probed at runtime — flash the single binary regardless of hardware. A missing chip shows a "not found" message instead of crashing. Run Hardware Self-Test to see what's detected.

What's different from upstream Bruce?

Same firmware base, plus genuinely-new bits: Hardware Self-Test (I²C+SPI probe), Si5351 live calibration + favorites, dual-NRF24 spectrum, CC1101 peak-hold spectrum, and a version-aware multi-board browser flasher.

Is this legal?

For authorized security testing and education only. Jammers and deauth against gear you do not own are illegal in most jurisdictions. You assume all responsibility. AGPL licensed.
