ESP32-S3 N16R8 — flash from your browser in one click

Your browser does not support Web Serial. Use Chrome or Edge (desktop). This page must be served over HTTPS.

Connect the device via the left USB-C (UART), press the button, pick the port.

How to

  1. Open in Chrome or Edge (desktop — Web Serial required).
  2. Connect the ESP32-S3 to the left USB-C port.
  3. Flash Device → select the serial port → Connect.
  4. Keep "Erase device" checked (clean install). Hit Install and wait.

⚠️ If flashing won't start: hold BOOT while connecting, or BOOT+RESET to enter download mode. The serial driver (CH34x / native USB) must be installed.
